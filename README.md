# 📝 Blog Platform Web App

A simple and interactive **Blog Platform** built using **HTML, CSS, and JavaScript**.  
This project allows users to create, view, and delete blog posts dynamically.

---

## 🚀 Features
- ✍️ Create and publish blog posts  
- 📄 Display posts instantly  
- ❌ Delete posts  
- 🎨 Clean and minimal UI  
- ⚡ Real-time updates (no page reload)  
- 📱 Responsive layout  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 📂 Project Structure

Blog-Platform/
│── Blog platform.html
│── README.md


---

## ▶️ How to Run
1. Download or clone this repository  
2. Open **Blog platform.html** in your browser  

---

## 📸 Overview
This app allows you to:
- Enter a blog title and content  
- Publish posts instantly  
- View all posts in a structured format  
- Delete posts when needed
